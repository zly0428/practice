Automation Technical Test Instructions

Hello, welcome to CSE.
As part of the Automation Developer candidate selection process, we are asking you to complete a Java/Selenium programming challenge to demonstrate your technical abilities.
If you are reading these instructions now then you have probably already executed the simple Java program was setup with Selenium dependencies. Currently downloaded are Chrome Driver and IE 32bit and 64bit Drivers. You can use the Driver/Browser combination of your choice.
Here are the next steps:
  • Create a folder with your name inside this project. All classes and files you create for this test should go into this folder. Feel free to make as many sub-folders, classes and files as you need.
  • Create a Java program to automate the CSE Store website. The site is here: http://perf1.csexperts.com:8080/softslate/
  • Test Data will come from the testcases/TestCaseInput.txt file, the format is: Test Case Name,Year,Item Description Search,Price Search
  • For each run create a new log file. Report any errors and exceptions to the log file with a timestamp.
  • The workflow for the test we will need to execute is as follows:
    1. Open the browser and navigate to the CSE Store.
    2. Click the 'Images by Year' link in the main content window or the Browse navigation sidebar. Your choice.
    3. Click the 'Images from XXXX' link, where the Year is given in the Test Case Input file. Assume that Year is mandatory and will always be provided in the file.
    4. Search for the first item that matches the Name and Price given in the Test Case Input file, and get the Page, Table Row and Column. For bonus get all the matches.
    5. For bonus do not click [View All], search for the matches on each page and also report the Page that matches were found on.
    6. For each Test Case, print a single row in the log file showing the Page, Table Row and Column that the matching item(s) were found.
      • Example test case: TestCase1,1993,Hubble Finds,
      • Example log report: Item found on Page 1 at Row[1] Column[2], Page 2 at Row[2] Column[1]
      • Bonus for extra information in log report
  • The Name can contain partial matches (i.e. "Hubble finds"). For bonus, allow the tester to input regular expressions into the Test Case Input file (i.e. "H[a-zA-Z]*\s+finds.*").
  • Price can be in dollars (102), or standard currency format (102.00)
  • Name and Price can be blank, in which case they can match any values
